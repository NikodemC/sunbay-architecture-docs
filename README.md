# Sunbay — dokumentacja architektury (Google Sheets)

Statyczna strona dla data engineera: jak Google Sheets są wpięte w obecną architekturę Sunbay
(zasilanie z softów księgowych → transformacja w arkuszu → import → downstream).

Otwórz `index.html` lokalnie albo zobacz wersję opublikowaną na GitHub Pages.

Sekcja „Transformacja w arkuszu" zawiera szablon do uzupełnienia o konkretne przykłady.
